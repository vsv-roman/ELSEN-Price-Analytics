# 🚀 ELSEN Price Dashboard — PWA

Progressive Web App для конкурентного ценового анализа.

## ✨ Возможности PWA

| Функция | Описание |
|---------|----------|
| 📱 Установка | На рабочий стол Android/iOS/Windows |
| 📡 Офлайн-режим | Работает без интернета |
| ⚡ Быстрая загрузка | Кэширование ресурсов |
| 🔄 Авто-обновления | Фоновая проверка обновлений |
| 📲 Pull-to-refresh | Потяните вниз для обновления |
| 🔔 Уведомления | Статус подключения |
| 🎨 Splash screen | Красивый экран загрузки |
| ⌨️ Горячие клавиши | Ctrl+Shift+R — обновление |
